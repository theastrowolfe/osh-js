Guide to contribute to the docs
-------------------------------

* Install Python version >= 2.7

* Install MkDocs
  > sudo pip install mkdocs

* cd to the osh-docs folder

* To preview the docs locally, run 
  > mkdocs serve

* The markdown source code is in the 'docs' folder
  The generated site is in the 'site' folder

* Site is generated  by running
  > mkdocs build


See MkDocs documentation for more details
http://mkdocs.readthedocs.org/en/stable/
